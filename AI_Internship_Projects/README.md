# 1-Month Artificial Intelligence Internship Projects

This repository contains the projects completed during my 1-Month Artificial Intelligence Internship.

## Projects Completed

### Customer Churn Prediction (Project 5)

A Machine Learning project that predicts whether a telecom customer is likely to churn (leave the service) based on customer demographics, account information, and service usage patterns.

Implemented Logistic Regression and Random Forest Classification models for churn prediction.

Achieved an accuracy of 81.62% using Logistic Regression.

---

### Twitter Sentiment Analysis (Project 2)

A Natural Language Processing (NLP) project that classifies tweets as Positive or Negative using Machine Learning techniques.

Utilizes text preprocessing, stopword removal, TF-IDF vectorization, and Logistic Regression for sentiment classification.

Achieved an accuracy of 75.62% on the Sentiment140 dataset.
